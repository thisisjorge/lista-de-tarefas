# Lista de Tarefas

Uma aplicação web simples para gerenciamento de tarefas diárias.

## Sobre o Projeto

Esta aplicação permite criar e gerenciar uma lista de tarefas de forma simples e intuitiva. Ideal para organizar atividades diárias, compromissos e lembretes.

## Funcionalidades

- Adicionar novas tarefas
- Marcar tarefas como concluídas
- Excluir tarefas
- Persistência de dados no navegador (localStorage)
- Interface limpa e responsiva

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript

## Como Usar

1. Clone o repositório:
```git clone https://github.com/thisisjorge/lista-de-tarefas.git```

2. Abra o arquivo `index.html` em seu navegador

3. Comece a gerenciar suas tarefas:
   - Digite uma tarefa no campo de entrada e pressione Enter ou clique no botão de adicionar
   - Clique em uma tarefa para marcá-la como concluída
   - Clique no ícone de exclusão para remover uma tarefa

## Demonstração

Acesse a demonstração online: [Lista de Tarefas](https://thisisjorge.github.io/lista-de-tarefas/)

## Recursos

- Design minimalista e intuitivo
- Salva automaticamente suas tarefas no navegador
